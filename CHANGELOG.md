# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-28

### Added
- Initial release of LangGraph4j
- Core graph-based workflow engine
- State management with flexible key-value store
- Node and Edge abstractions
- Conditional edges and dynamic routing
- Human-in-the-loop support
- State persistence (InMemoryStateStore, FileStateStore)
- Event listener system for workflow monitoring
- Loop detection and max iteration protection
- Graph visualization utilities (Mermaid, DOT)
- Comprehensive examples (Approval, DataPipeline, AIAgent)
- Full test coverage
- Documentation and README

### Features
- Type-safe workflow definitions with Java generics
- Fluent builder API for graph construction
- Support for complex branching logic
- Checkpoint and resume capabilities
- Minimal dependencies (only SLF4J)
